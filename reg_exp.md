# 정규표현식

### 기호에 따른 의미

| 기호 |  |
| :--- | :--- |
| \d | 0-9 |
| \D | 0-9를 제외한 나머지 |
| . | 아무거나 |
| \. | Dot, Period |
| \[abc\] | Only a, b or c |
| \[^abc\] | Not a, b nor c |
| \[a-z\] | a-z |
| \[0-9\] | 0-9 |
| \w | 영어와 숫자 |
| \W | 영어와 숫자를 제외한 나머지 |
| {m} | m번 반복 |
| {m,n} | m~n번 반복 |
| \* | 0번 이상 |
| + | 1번 이상 |
| ? | 있거나 없거나 \(0 or 1\) |
| \s | Whitespace \(Space, tab, new line, carriage return\) |
| \S | Whitespace를 제외한 나머지 |
| ^a | a로 시작 |
| a$ | a로 끝 |
| \(\) | Grouping |
| \(abc\|def\) | abc or def |
